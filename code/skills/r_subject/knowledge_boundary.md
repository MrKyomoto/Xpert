# 技能：学科内容边界

限定 C/E 审查范围，避免重复归因和越界评价。

## 不负责

- 不评价 A/B/D 结构、环节展开、任务链或目标—活动映射。
- 不评价 F1—F6，不判定学段课标素养名称是否匹配。
- 不执行最终评分，不决定停止迭代，不仲裁专家冲突。

## 归因与转交

- 将事实、数据、量纲或安全含糊归 C；将纯语言含糊、冗余、术语漂移或编号混乱归 E。
- 将课标素养与学段语言问题转交 `r_literacy`。
- 将结构、流程、任务链和一致性问题转交 `r_learner`。
- 将冲突转交 `r_chair`，将最终分数转交 `r_judge`。

## 证据边界

- 不将“可能有错”当作已确认错误；对需特定学科专业验证的结论标注“待教研核对”。
- 不把同一原文缺陷同时写入 C 和 E。
