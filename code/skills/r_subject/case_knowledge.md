# 技能：PBL 学科证据案例方法卡

仅提取附录 B 案例的证据、校验与安全方法，用于 PBL 课的 C/E 审查。

## 可迁移方法

- **案例 1（化学·应急蜡烛的设计与制作）**：先将产品性能转换为可检验指标，再拆分变量、设置对照、记录证据并统整验证；同时单列燃烧、器材和操作安全。
- **案例 2（数学·设计一款皮筋测力计）**：使用重复测量、图表表征、模型拟合、适用范围检验和异常数据处理，让结论可复核。
- **案例 3（生物·共建清新校园——防治校园飞絮）**：将现场调查、因果假设、多源证据、可测性/可证性/可控性筛选和局部试行连成证据链。
- **案例 4（语文·如果山水会说话）**：使文本解读、景物特征、情感判断和写作策略都能回到具体文本证据，并核对受众、目的与语体一致。
- **案例 5（环境·我是绿色农药研发师）**：分别验证效果、非目标影响、环境风险和使用安全，使方案修改由测试证据驱动。

## 使用限制

- 只迁移变量控制、多源证据、安全核对、模型验证和迭代校验方法。
- 不复制案例的同源答案、精确公式结果、实验数据、材料配比或操作参数。
- 若当前课题与案例同源，只能用本卡的抽象方法，不得还原案例内容。
