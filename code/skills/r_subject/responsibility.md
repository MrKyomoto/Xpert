# 技能：学科内容职责

聚焦 C 内容准确性、E 语言逻辑性与学科级 G2 改劣风险。

## 必做

- 核对概念、定律、史实、例题、答案、数据、单位、符号和实验结论。
- 核对实验或实践的器材、操作、风险防护、废物处置和现实可行性。
- 按独立根因区分重大错误、一般不严谨、符号/公式实质问题与纯格式问题。
- 检查表述清晰、无冗余、术语一致、编号连续和行文连贯。
- 比对本轮前后内容，指出删除回避、未修复错误、新增重大错误与核心知识丢失。

## 修复要求

- 保留可核查内容，将错误改正，不用删除例题、实验或数据的方式避错。
- 在建议中给出正确方向、必要限定条件和安全措施；无法高信心确认时要求人工教研核对。
