# 技能：C/E 量规映射

按附录 A 的 C/E 条款分类根因，只提修改意见，不代替 Judge 结算总分。

## C 内容准确性

- **重大知识性错误**：标记概念、定律、例题答案或危险操作的独立根因；附录 A 的 C 原始维度分每根因扣 2。
- **一般性不严谨**：标记学科事实含糊、量纲缺失或史实细节偏差；每根因扣 0.5。
- **符号/公式实质问题**：标记量纲错误、符号前后不一致或公式乱码致不可读；每根因扣 0.5。
- **格式合规问题**：仅在不涉及内容正确性时标记；原始维度分每次扣 0.5，全文合计最多扣 1。
- **根因单元化**：合并同根因的连锁后果和全文重复；对同段的独立根因分别记录。
- **内容回避**：检查是否至少保留一项可核查的完整例题、实验方案或具体数据参数；低于底线时 C 原始维度分封顶 3。

## G2 改劣检查

- 将未修复和删除回避按原错误档位处理。
- 将输出新增的重大错误标为双倍扣分风险；附录 A 规定每根因扣 4 个 C 原始维度分。
- 将核心知识或关键例题丢失同时提交 Chair 做 G1 保真复核。

## E 语言逻辑性

- **E1**：消除纯语言层面的歧义。
- **E2**：删减重复段落和车轱话，不在 B 重复惩罚。
- **E3**：统一同一概念的术语。
- **E4**：修复编号跳档与层级混乱。
- **E5**：使环节间过渡和逻辑关系清晰。
