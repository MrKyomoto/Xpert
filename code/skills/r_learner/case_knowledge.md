# 技能：PBL 任务链案例方法卡

仅提取附录 B 案例的可迁移任务链方法，用于 PBL 课的 A/B/D 审查。

## 可迁移方法

- **案例 1（化学·应急蜡烛的设计与制作）**：使任务链遵循“明确性能目标—分析影响因素—分项探究—统整设计—测试迭代—公开交流”，让各阶段产出相互依赖。
- **案例 2（数学·设计一款皮筋测力计）**：使任务链遵循“重建问题解决原理—采集与分析数据—建立模型—形成原型—按场景优化—答辩反思”。
- **案例 3（生物·共建清新校园——防治校园飞絮）**：使任务链遵循“现场调查—成因分析—方案设计—局部实施—证据评估—迭代发布”，使真实反馈返回下一轮。
- **案例 4（语文·如果山水会说话）**：使任务链遵循“明确受众与交际目的—研读范例—归纳策略—创作产出—受众反馈—修订展示”。
- **案例 5（环境·我是绿色农药研发师）**：使任务链遵循“调查需求—筛选方向—设计方案—测试效果与风险—优化方案—面向真实受众发布”。

## 使用要求

- 为每个任务写明对上一任务产出的使用、学生主活动、阶段成果、检核和反思。
- 只迁移任务链结构和衔接方法，不固定任务数量或活动时长。
- 不复制案例的同源答案、成品方案、实验步骤、精确参数或数值门槛。
- 若当前课题与案例同源，只能用本卡的抽象方法，不得还原案例内容。
