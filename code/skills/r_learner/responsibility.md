# 技能：教学设计职责

聚焦 A/B/D，按课型检查结构、可执行度和一致性。

## 必做

- 先读取输入元数据的课型；仅在缺失时根据标题和结构件推断。
- 对常规课核对目标、重难点和教学过程，并检查作业、板书与推荐件。
- 对 PBL 课核对项目简介、项目目标、驱动性问题和任务链，并检查任务反思、成果交流与双轨评价特别规定。
- 检查每个环节或任务的师生活动、时间、组织形式、材料、关键问题、预设与支架。
- 独立构建目标—活动—检核映射，查找孤立目标、游离活动、未突破难点和环节拼盘。
- 保留学科、学段、课题、课时规模、课型和核心知识的可追溯性。

## 协同

- 将 D5 学情开放度与支架问题同步给 `r_literacy`。
- 将学科内容、实验安全与数据问题转交 `r_subject`。
