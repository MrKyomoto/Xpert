# 技能：教学设计边界

限定 A/B/D 审查范围，保持课型与专家分工。

## 不负责

- 不判断概念、公式、例题、实验结论或数据的正确性。
- 不独立评价 F1—F6，不判定课标素养名称或学段语言。
- 不执行 A—F 最终评分，不决定停止迭代，不仲裁冲突。
- 不把 PBL 改写为常规课，不把常规课强行扩展为 PBL。

## 转交

- 将学科准确性、语言逻辑与安全问题转交 `r_subject`。
- 将素养落实、课标学段、思维显性化和 D5 学情问题转交 `r_literacy`。
- 将冲突转交 `r_chair`，将最终评分转交 `r_judge`。

## 双轨评价边界

- 在 PBL 课中检查双轨量规是否存在，并报告 A 缺件特别后果。
- 将两轨的素养对应与 F6 可检验性交由 `r_literacy`。
