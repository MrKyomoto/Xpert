# 技能：Chair 职责

将专家提案转换为可追溯决策、冲突记录和完整教案。

## 必做

- 核验每条提案的位置、问题、依据、建议和来源角色。
- 对重复提案去重，对互补提案合并，对不相容提案识别冲突。
- 为每个 prompt 已提供的 `Pxxx` ID 记录 `accepted/merged/rejected` 决策。
- 核对 G1 保真：保持学科、学段、课题、课型，控制课时变化，保留核心知识与关键例题的可追溯性。
- 将获接受或合并的修改落到教案相应位置，保持全文术语、编号和环节衔接一致。
- 输出完整教案，并使决策摘要与实际终稿一致。

## 不负责

- 不执行 A—F 评分或 Judge 的停止建议。
- 不在缺少专业依据时自行创造学科结论、课标解释或精确数据。
