# 技能：教案质量六维量规

使用附录 A v1.1 的唯一计分口径：先得 `[0,5]` 原始维度分，再换算加权分。

## A 结构完整性（权重 10）

- 对 A1 核心件、A2 非核心件、A3 结构实质性分别判 0/1/2，再按 `5×点数和/6` 计算原始维度分。
- **常规课核心件只有**：教学目标；教学重难点；教学过程（含情境导入、探究学习、例题巩固、课堂小结或同义环节）。
- 常规课非核心件为作业布置、板书设计；学情分析、教材分析、教学反思每个有实质内容可为 A 原始维度分加 0.5，最终不超过 5。
- **PBL 核心件只有**：项目简介（背景与价值）；项目目标（含素养目标）；驱动性问题；任务链（项目—任务—活动分级且有衔接说明）。
- PBL 非核心件为每任务的总结反思、成果交流设计。
- **PBL 特别规定**：缺双轨评价设计时同时触发 A 缺件与 F6 = 0；不把双轨评价写成上述四个核心件之一。
- 任一课型缺任一核心件时，A 原始维度分封顶 2，即 A 加权分封顶 4/10。缺件判定前先核对同义标题和实质内容。

## B 内容丰富性（权重 15）

- 对 B1 师生双边活动展开度、B2 时间/组织/材料可执行性、B3 本课关键问题的预设与应对、B4 与主任务耦合的支架分别判 0/1/2。
- 按 `5×点数和/8` 计算原始维度分；评可执行性密度，不评篇幅。
- 将可跨课复用的预设和支架至多判为部分满足。

## C 内容准确性（权重 20）

- 从 5 个原始维度分起扣：重大知识错误/危险操作每独立根因扣 2；一般不严谨每根因扣 0.5；符号/公式实质问题每根因扣 0.5；纯格式问题每次扣 0.5、全文最多扣 1。
- 合并同根因连锁后果与全文重复，分别记录互相独立的根因。
- 若无至少一道带完整过程的例题、完整实验方案或具体数据参数（按学科性质取其一），C 原始维度分封顶 3，即 C 加权分封顶 12/20。
- G2 中新增重大错误每根因扣 4 个 C 原始维度分；删除回避按原错误档位处理。

## D 内容一致性（权重 15）

- 对 D1 目标—活动映射、D2 重难点呼应、D3 环节产出的实质衔接、D4 作业/板书一致、D5 与官方学情 profile 的匹配分别判 0/1/2。
- 按 `5×点数和/10` 计算原始维度分。
- 独立建立映射证据；不把教案自带映射或“本环节落实目标”作为证据。

## E 语言逻辑性（权重 10）

- 对 E1 表述清晰、E2 无重复冗余、E3 术语一致、E4 编号层级规范、E5 行文连贯分别判 0/1/2，按 `5×点数和/10` 计算原始维度分。
- 只在 E 惩罚冗余；将学科事实、数据和量纲含糊留在 C。

## F 素养导向性（权重 30）

- 对 F1 素养目标行为化、F2 学段课标符合、F3 成体系思维参与、F4 非装饰情境与整合性主活动、F5 学生主体与生成、F6 目标检核可执行性分别判 0/1/2。
- 按 `5×点数和/12` 计算原始维度分。
- 若无活动证据的素养条目占比超过三分之一，F 原始维度分封顶 3；若全部无证据，封顶 2。
- 只将“表达思路且有教师预设应对或后续使用”视为合格思路外显。
- 只将导入后至少两个环节实质使用的情境视为非装饰性情境。
- PBL 额外检查：专门的思路构建/表达环节；驱动问题与目标绑定、复杂、开放且可行；任务拆解自然展开问题解决思路；每任务有整合性主活动；个人成长成果与问题解决成果分轨评价且水平描述绑定本项目。
