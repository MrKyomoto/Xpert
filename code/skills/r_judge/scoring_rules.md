# 技能：加权计分与停止建议

按附录 A 计算连续分值，并将最终停止控制权留给编排代码。

## 计分

1. 计算 A/B/D/E/F 各子指标的 0/1/2 点数和。
2. 使用 `raw_A=5×ΣA/6`、`raw_B=5×ΣB/8`、`raw_D=5×ΣD/10`、`raw_E=5×ΣE/10`、`raw_F=5×ΣF/12`。
3. 使用 `raw_C=clamp(5-Σ扣分,0,5)`，并执行内容回避与 G2 规则。
4. 应用 A 核心件封顶、A 推荐件加分、C 内容回避封顶、F1 贴标签封顶和 PBL 双轨评价特别规定。
5. 换算 `A=raw_A/5×10`、`B=raw_B/5×15`、`C=raw_C/5×20`、`D=raw_D/5×15`、`E=raw_E/5×10`、`F=raw_F/5×30`。
6. 使用未取整的连续值求和；确保 `total_score=A+B+C+D+E+F`。

## `should_stop` 建议

- 仅当同时满足以下三项时输出 `true`：当前总分不低于 85；存在上一轮有效总分；当前总分相比上一轮提升小于 2。
- 首轮、无上轮有效分数或当前总分低于 85 时必须输出 `false`。
- 将该字段仅视为 Judge 建议；编排代码必须使用同一三条件重新计算，并作出最终停止决定。
