# 技能：素养量规与课标映射

按附录 A 的 F1—F6 和附录 C 的学段语言查找可引证的问题。

## F1—F6 检查

- **F1 素养目标行为化**：要求正确课标名称和“能做什么”的行为同时出现；检查活动的认知要求，不把关键词回声当作落实。
- **F2 学段课标符合**：核对学科、学段、主题或大概念；把混用义教/高中名称视为学段失位风险。
- **F3 思维参与**：要求递进问题链和合格的思路外显；孤立的“说说怎么想”不算证据。
- **F4 真实情境与探究**：要求导入后至少两个环节实质使用情境，并设置综合多种知识或方法的主活动。
- **F5 学生主体与生成**：要求学生自主组织路径、产出可观察成果，并根据差异产生讨论或迭代。
- **F6 目标达成检核**：要求检核内容对应本课目标，且水平描述能被本课活动实际观察；无测量活动支撑的数值不作充分证据。

## PBL 加强检查

- 检查 F3 是否有专门环节构建并表达问题解决思路。
- 检查 F4 的驱动性问题是否紧扣目标、具有复杂度与开放性且可行；使任务拆解自然展开问题解决思路。
- 检查 F6 是否分别设置“个人成长成果”和“问题解决成果”量规，且两轨水平描述绑定本项目；缺失时明确标注“F6 = 0，并触发 A 缺件”的特别规定。

## 学科 × 学段语言

- **数学义教**：使用“三会”总纲及对应学段的数感、量感、符号意识、运算能力、几何直观、空间观念、推理、数据、模型、应用与创新表述；**数学高中**：数学抽象、逻辑推理、数学建模、直观想象、数学运算、数据分析。
- **物理义教/高中**：均为物理观念、科学思维、科学探究、科学态度与责任；高中需用与课题匹配的下位要素体现深度。
- **化学义教**：化学观念、科学思维、科学探究与实践、科学态度与责任；**化学高中**：宏观辨识与微观探析、变化观念与平衡思想、证据推理与模型认知、科学探究与创新意识、科学态度与社会责任。
- **生物学义教**：生命观念、科学思维、探究实践、态度责任；**生物学高中**：生命观念、科学思维、科学探究、社会责任。
- **语文义教**：文化自信、语言运用、思维能力、审美创造；**语文高中**：语言建构与运用、思维发展与提升、审美鉴赏与创造、文化传承与理解。

## D5 学情检查

- 检查任务难度、开放度、先备知识、支架与分层是否明确呼应 profile。
