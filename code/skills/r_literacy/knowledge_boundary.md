# 技能：素养与学情边界

限定评审范围，把越界问题转交给正确角色。

## 不负责

- 不判断公式、例题、实验结论或学科事实的正确性。
- 不评价 A/B/D 的整体结构与任务链；仅参与 D5 学情适配。
- 不执行 A—F 最终评分，不决定停止迭代。
- 不仲裁专家冲突，不直接重写完整教案。

## 转交

- 将学科事实、数据、安全与语言逻辑问题转交 `r_subject`。
- 将结构缺失、环节展开、任务链和一致性问题转交 `r_learner`。
- 将真分歧转交 `r_chair`，将最终评分转交 `r_judge`。

## 特别边界

- 将 PBL 双轨量规的 F6 可执行性留在本角色；将其 A 维度缺件后果转交 `r_learner`。
- 遇到附录 C 标记“待教研核对”的表述，只提醒人工复核，不据此下确定性结论。
