# 技能：PBL 素养案例方法卡

仅提取附录 B 案例的可迁移方法，用于 PBL 课的素养与学情审查。

## 可迁移方法

- **案例 1（化学·应急蜡烛的设计与制作）**：赋予学生真实任务身份，让模型建构、实验证据、迭代反思和公开成果共同支撑素养。
- **案例 2（数学·设计一款皮筋测力计）**：让学生经历“现实问题—建模—验证—修正—表达”，并将个人学习证据与产品成果分开记录。
- **案例 3（生物·共建清新校园——防治校园飞絮）**：从学生熟悉场域中的真问题出发，让调查证据、社会责任、实践行动与持续迭代相互连接。
- **案例 4（语文·如果山水会说话）**：明确作者身份、真实受众、发布场景和交际目的，让语言实践、审美选择与文化理解落到可观察产出。
- **案例 5（环境·我是绿色农药研发师）**：分别评价问题解决成果与个人成长成果，并将协作、反思、生态责任和证据驱动迭代纳入检核。

## 使用限制

- 只迁移“身份—受众—产出—证据—反思—评价”等方法。
- 不复制案例的同源答案、成品文案、实验配方、精确参数或数值门槛。
- 若当前课题与案例同源，只能用本卡的抽象方法，不得还原案例内容。
