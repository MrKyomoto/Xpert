# 技能：素养与学情职责

聚焦 F1—F6 和 D5，把课标与学情转换为可执行的修改建议。

## 必做

- 核对 F1：使素养目标同时使用正确学段语言和可观察行为。
- 核对 F2：判断素养与学科、学段、主题或大概念的深度是否匹配。
- 核对 F3：要求学生表达“怎样想到”或“为何这样设计”，并检查教师应对或后续使用。
- 核对 F4：检查真实情境是否在导入后的多个环节继续承载任务。
- 核对 F5：为学生保留组织路径、方法选择、成果产出和生成性讨论空间。
- 核对 F6：使每条检核可由本课实际活动观察；PBL 课核对双轨量规。
- 核对 D5：以输入的官方学情 profile 为准，检查难度、开放度与支架。

## 依据

- 使用附录 A 的 F 维度与 D5 条款。
- 使用附录 C 中对应“学科 × 学段”的语言。
- 仅在 PBL 课中使用 `case_knowledge` 的可迁移方法。
